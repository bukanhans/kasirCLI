# <h1 align="center">Kasir CLI (Command Line Interface)</h1>
## Aplikasi CLI yang di kembangkan menggunakan python dan menggunkana Mysql sebagai penyimpanan data yang akan di gunakan dalam aplikasi

## Fitur Aplikasi
- Dapat melakukan Create, Read, Update dan Delete di dalam aplikasi
- Transaksi dilakukan dengan mengambil data harga di database dan di kali dengan jumlah yang di inputkan user

## Teknologi yang digunakan
- Python
- Mysql

## Cara menjalankan
clone repositori ini
```bash 
git clone https://github.com/bukanhans/kasirCLI
```
jalankan XAMPP dan buat database (nama opsional)
lalu import file db_kasir.sql dan sesuaikan
lalu setup database di dalam file db.py 

lalu jalankan main.py 
```python
python main.py
```
